### king

#### 后台支持

goadmin
```bash
go get github.com/GoAdminGroup/go-admin@latest 
```
详细文档见 [http://doc.go-admin.cn/](http://doc.go-admin.cn/)

#### swagger支持

```bash
// 需要安装依赖
go get -u github.com/swaggo/swag/cmd/swag
// 生成swagger docs
swag init
// 暂时swagger浏览器地址
http://127.0.0.1:9033/swagger/index.html
```
